# PromoDash

PromoDash is a Python-based application designed to aggregate and deliver small business-focused promotions, discounts, and deals directly to subscribers. It serves as a centralized hub for identifying opportunities that can benefit startups and growing enterprises.

## Features

- **Automated Scraping:** Continuously monitors targeted websites and platforms for new deals and promotions.
- **Personalized Alerts:** Matches deals to specific business profiles based on industry, size, and other criteria.
- **Multi-channel Delivery:** Sends notifications directly via email and WhatsApp integration.

## Technologies Used

- **Language:** Python
- **Libraries:** BeautifulSoup, Scrapy, pandas

## Installation

1. Clone the repository to your local machine:

```bash
git clone https://github.com/your-username/PromoDash.git
```

2. Navigate into the project directory:

```bash
cd PromoDash
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```
